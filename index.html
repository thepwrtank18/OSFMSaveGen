OSFM Save Generator
This generates a post Solstice save file for OneShot, for those who lost their save but want to play OneShot: Fading Memory without replaying the base game.
Do not use if you haven't beaten Solstice yet, since you'd be missing a ton of context.
Also, you may consider replaying the base game with OSFM installed anyway, since there are subtle easter eggs you might find.
